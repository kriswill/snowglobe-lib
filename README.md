# Getting Started

Navigate to https://cache.earthgman.dev/nixos-installers and download the ISO image for your CPU arch.

Supported Architectures:
- x86_64-linux
- aarch64-linux

Note: Installers suffixed with -small will be stripped of linux firmware modules.
This version may cause some hardware devices to not function properly.
When in doubt just use the default.

You can also build / modify the iso images yourself. [TODO link]

Once you boot the installation medium the rest should be self explanitory.

I encourage you to report issues if you have any.
